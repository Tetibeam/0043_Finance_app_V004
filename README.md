# 📘 Personal Finance Intelligence Dashboard

次世代型パーソナルファイナンス・プラットフォーム

あなたの資産・収支・投資・リスク・FIRE 進捗を一つの画面で可視化する総合 Wealth Management Dashboard です。

## 🚀 Features

### ワンビュー財務管理

- 総資産、投資リターン、キャッシュフロー、貯蓄率、FIRE 進捗などを 1 ページで即把握
- KPI カード＋高解像度チャート構成で状況判断を高速化

### 高度な資産配分分析

- 資産カテゴリ別の比率を自動集計
- リバランスの判断に使える安定した可視化

### 収入／支出の構造分析

- 一般支出と特別支出を分離
- 必要生活費、変動費、浪費の構造を俯瞰し改善点を抽出

### 投資パフォーマンスの深掘り

- TWR / MWR、資産別寄与度、リスク比率など高度な分析
- 銘柄ごとの成績比較で改善ポイントを特定

### FIRE 進捗トラッキング

- 時系列の達成率を表示
- 目標資産シミュレーションとの乖離を常時モニタリング

### リスク・安全性評価

- 防衛資金、負債比率、生活コスト耐性などを定量化
- 家計の「耐久力」を可視化

### シナリオシミュレーション

- 投資リターン・収入・支出のシナリオを自由設定
- 将来の純資産推移を予測
- 2 年分のデータ蓄積で AI 学習を用いた自動予測も実装可能

## 🎨 Page Structure

### 1. Portfolio Command Center

- **目的**:
  ユーザーの財務状況を 1 ページで俯瞰する“司令塔”。資産推移、収支、投資成果、FIRE 進捗などの主要 KPI をまとめて確認し、意思決定の初動を素早く行えるようにする。
- **構成**:
  - **ヘッダー**
    - 最新日付、純資産進捗率、純資産額、目標額、差異を記載
  - **メインコンテンツ**:
    - **FIRE Readiness**:
      - 目標資産額に対する達成率の推移を日別で示すグラフ
    - **Savings Efficiency**:
      - 月間で収入に対してどれだけ資産形成に回せているかを示すグラフ
    - **Net Worth Trajectory**:
      - 純金融資産の推移を目標と実績で示すグラフ
    - **Portfolio Performance**:
      - ポートフォリオの推移を目標と実績で示すグラフ
    - **Cash Flow – Routine**:
      - 月間の一般収支の現金フローを目標と実績で示すグラフ
    - **Cash Flow – Exceptional**:
      - 月間の特別収支の現金フローを目標と実績で示すグラフ

### 2. Allocation Matrix

- **目的**:
  資産構成を可視化し、ポートフォリオのバランス・偏り・リスク許容度との整合性を判断する。
  リバランスの必要性を直感的に理解できる。
- 構成:

  - ヘッダー
  - **メインコンテンツ**:
    - **Interactive Treemap**:
      - 資産構成の全体像。サンバーストチャート or ツリーマップ
    - **Target Deviation**:
      - 「理想」と「現実」の差分グラフ。「日本株 +5.2% (Overweight)」などを表示。
    - **Portfolio Efficiency Map**:
      - 各資産クラス（株式、債券など）のリスク（ボラティリティ）とリターンを散布図で表示します。
      - ユーザーが取っているリスクと、それに見合うリターンが得られているかを資産クラスごとに判断する
    - **Liquidity Pyramid**
      - 現金化しやすさの資金がしっかりあるかを確認する
    - **True Risk Exposure Flow**:
      - 通貨と投資対象地域をサンキーダイアグラムで表示（通貨と投資対象国の流れみて、認識違いがないように）
    - **Rebalancing Workbench**:
      - 「リバランスシミュレーター」。スライダーを動かして売買後の配分をプレビューする機能。

### 3. Cashflow Analytics

- **目的**:
  日常の収入・支出の動きを分析し、可処分所得・貯蓄余力・固定費比率などを把握する。
  長期的な資産形成に向けて、改善ポイントを明確化する。

### 4. Extraordinary Events

- **目的**:
  臨時収入や突発的な支出（耐久家財、医療費、旅行費など）を切り分けて分析し、
  “通常の生活コスト” と “イベント的支出” を分離して評価する。
  生活習慣の改善と、長期的な支出計画の両方を整理できる。

### 5. Investment Performance Lab

- **目的**:
  投資ポートフォリオのリターン、ボラティリティ、最大ドローダウン、資産別寄与度などを詳細に解析する。
  “どの投資が成果を出しているのか” を科学的に理解し、改善と最適化につなげる。

### 6. Risk & Stability Monitor

- **目的**:
  生活防衛資金、負債比率、収入安定性、キャッシュバッファなど “守りの指標” を評価する。
  景気後退やライフイベントに耐えうる財務安全性をチェックし、
  ファイナンシャル・リスクマネジメントを強化する。

### 7. FIRE Progress Tracker

- **目的**:
  現在の純資産が FIRE（経済的独立）の目標にどれだけ近づいているかを追跡。
  達成率、ギャップ、改善ペースを可視化し、中長期目標の達成をナビゲートする。

### 8. Scenario Engine

- **目的**:
  収入・支出・投資リターンのシナリオを変更しながら将来の資産推移を予測。
  “どの戦略を選べば最短で FIRE に到達できるか” といった長期シミュレーションを支援する。
  （将来的には AI 学習による自動予測も実装可能）
